This plugin allows you to display and sell Best Buy products on your blog. An affiliate account lets you make commission on the products you sell.

== Description ==

The Best Buy Products Plugin allows you to automatically display Best Buy products on your WordPress site. Integrated with the [Best Buy Affiliate Program](https://bbyopen.com/affiliate-program "Best Buy Affiliate Program"), with an affiliate ID you can earn commissions on the products you sell. Simply enter the items you want to display, your Best Buy API key, and the product images will be displayed with a link to the product detail page on BestBuy.com. You'll be eligible to earn commissions on any purchase referral you generate. Additional detail on Best Buy APIs can be found at [bbyopen.com/developer](https://bbyopen.com/developer/)

== Installation ==

1. Upload `best-buy-products-plugin` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Apply for a [BBYOpen API key](https://bbyopen.com/developer "BBYOpen API").
1. Optionally, you can also apply for a [Best Buy Affiliate Account](https://bbyopen.com/affiliate-program "Best Buy Affiliate Account").
1. In Appearance->Widgets add the Best Buy Products widget to a widget area.
1. Configure your plugin, enter your API Key, and optionally enter your affiliate key.

== Frequently Asked Questions ==

= Why do I need to sign up for an API key? = 

The API key grants you access to the Best Buy product information.

= How do I sign up for an API key? =

You can sign up for an API key at the [Best Buy BBYOpen Developer Portal](https://bbyopen.com/developer)

= Why do I need to sign up for an Affiliate Account? =

The Affiliate Program allows us to pay you for the commissions you earn by selling Best Buy products.

= How do I sign up for an Affiliate Account? =

To sign up for an Affiliate Account, follow the instructions on the [Best Buy BBYOpen Developer Portal](https://bbyopen.com/affiliate-program).
